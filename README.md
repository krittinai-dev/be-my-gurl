# 💕 be-my-gurl — ขอเป็นแฟนหน่อยครับ ปลื้ม

เว็บไซต์สุดโรแมนติกสำหรับขอเป็นแฟน

## ฟีเจอร์

- 💌 **Intro** — ข้อความพิมพ์ออกมาช้าๆ แบบ typewriter
- 💖 **Reasons** — แสดงเหตุผลที่ชอบทีละข้อ พร้อม animation
- 🌹 **The Question** — คำถามสำคัญ พร้อมปุ่ม "ใช่" และ "ไม่ใช่"
- 🏃 **Runaway No** — ปุ่ม "ไม่ใช่" วิ่งหนีเวลาเอาเมาส์ไปแตะ
- 🎉 **Celebration** — confetti และวันที่เริ่มต้นของเรา

## วิธีใช้

เปิด `index.html` ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม

## ปรับแต่ง

แก้ค่าใน `scripts/main.js` ที่บล็อก `CONFIG`:

```js
const CONFIG = {
  introText: 'ถึง ปลื้ม ที่ผมแอบชอบ...',  // ข้อความเปิด
  reasons: [                                  // เหตุผลที่ชอบ
    { icon: '😊', text: '...' },
    ...
  ],
  noEscapeMessages: [ ... ],                  // ข้อความเมื่อปลื้มพยายามกดไม่ใช่
};
```

## Structure

```
be-my-gurl/
├── index.html
├── styles/
│   └── main.css
├── scripts/
│   └── main.js
└── assets/        ← ใส่รูปภาพได้ที่นี่
```
